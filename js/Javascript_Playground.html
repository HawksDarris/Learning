jQuery

The basic syntax:
$(selector).action()

Everything in jquery starts with a $ like so:

$(this).hide() hides this
$("p").hide() hides all <p>
$(".test").hide() hides all in a class called test
$("#test").hide() hides an id called test

As you can see, it is using CSS notation for the selector.

Making sure the code doesn't run until the page is loaded.

jQuery has a function called ready() that takes care of that by waiting for all HTML elements to be loaded before executing.

Just put the code inside a ready() function:
$(document.ready(function() {
alert("Some message");
});
